# Query Strategy Reference

## Effective German Pharma Query Patterns

This reference provides guidance on formulating high-quality Perplexity searches for German pharmaceutical market research. These are patterns and approaches, not prescriptive templates.

### General Principles

1. **Specificity over breadth** - Narrow queries yield better data
2. **Include year/timeframe** - Ensures currency of information
3. **Use German medical terminology** - Improves relevance of German sources
4. **Search incrementally** - Validate with follow-up searches

### Query Pattern Categories

#### Epidemiology & Prevalence
**Goal:** Find patient population size, disease burden, incidence/prevalence rates

**Effective patterns:**
- Disease + "Prävalenz" + Country + Year
- Disease + "Anzahl Patienten" + Country + "Statistik"
- Disease + "Inzidenz" + "Trend" + Timeframe

**Example searches:**
- "Typ-2-Diabetes Prävalenz Deutschland 2024"
- "Anzahl Psoriasis-Patienten Deutschland RKI"
- "Multiple Sklerose Inzidenz Trend 2020-2024"

#### Market & Competition
**Goal:** Understand competitive landscape, market share, prescribing patterns

**Effective patterns:**
- Drug + "Marktanteil" + Country + Competitor
- Therapeutic class + "Wettbewerb" + Country
- Drug + "Verschreibungszahlen" + Year

**Example searches:**
- "Biologika Psoriasis Marktanteil Deutschland Vergleich"
- "GLP-1 Agonisten Wettbewerb Deutschland 2024"
- "Humira Verschreibungszahlen Deutschland 2023"

#### Prescriber & Specialists
**Goal:** Identify which specialists treat the condition, practice patterns

**Effective patterns:**
- Specialty + Disease + "Anzahl" + Country
- Specialty + "Verschreibungsverhalten" + Drug
- Disease + "Versorgung" + "Fachärzte"

**Example searches:**
- "Dermatologen Psoriasis Behandlung Deutschland Anzahl"
- "Diabetologen Verschreibungsverhalten GLP-1"
- "Multiple Sklerose Versorgung Neurologen Praxen"

#### Market Access & Reimbursement
**Goal:** Understand pricing, AMNOG assessment, reimbursement status

**Effective patterns:**
- Drug + "AMNOG" + "Nutzenbewertung"
- Drug + "Erstattung" + "GKV"
- Drug + "Preis" + Country + Year

**Example searches:**
- "Skyrizi AMNOG Nutzenbewertung IQWiG"
- "Ozempic Erstattung GKV Deutschland"
- "Dupixent Preis Deutschland 2024"

### Advanced Techniques

#### Temporal Comparison
To understand trends, formulate paired searches:
- Search 1: "[Topic] [Year 1]"
- Search 2: "[Topic] [Year 2]"
- Compare results to quantify change

#### Source Triangulation
When data conflicts, search for multiple authoritative sources:
- Official statistics (Destatis, RKI)
- Professional associations (specialty societies)
- Payer data (AOK, Barmer reports)

#### Validation Searches
After finding a key statistic, validate with:
- "[Statistic] [Year] [Original source name]" - Confirm the number
- "[Topic] aktualisierte Zahlen" - Check for newer data
- "[Topic] Quelle Destatis/RKI" - Find official confirmation

### Common Pitfalls to Avoid

1. **Overly broad queries** - "Diabetes Deutschland" → Too general
   - Better: "Diabetes Typ 2 Prävalenz Deutschland 2024"

2. **Missing year/timeframe** - "Biologika Marktanteil"
   - Better: "Biologika Marktanteil Psoriasis 2024"

3. **English terms in German context** - "Market share Diabetes Germany"
   - Better: "Marktanteil Diabetes Medikamente Deutschland"

4. **Vague data requests** - "How common is disease X?"
   - Better: "Prävalenz [Disease X] Deutschland pro 100.000"

### Source-Specific Search Patterns

#### For Official Statistics (Destatis, RKI, KBV)
Add source name to ensure authoritative data:
- "[Disease] Prävalenz RKI"
- "[Specialty] Anzahl KBV Statistik"
- "[Treatment] Versorgung Destatis"

#### For Guidelines (S3-Leitlinien, AWMF)
Include guideline terminology:
- "[Disease] S3-Leitlinie AWMF"
- "[Disease] Behandlungsleitlinie Deutschland"
- "[Specialty society abbreviation] Leitlinie [Disease]"

#### For AMNOG/G-BA Information
Use official terminology:
- "[Drug name] Nutzenbewertung G-BA"
- "[Drug name] Zusatznutzen IQWiG"
- "[Drug name] Erstattungsbetrag GKV"

### Search Sequencing Example

**Research task:** Analyze German market for Atopic Dermatitis and Dupixent

**Search sequence:**
1. "Neurodermitis Prävalenz Deutschland 2024" (epidemiology)
2. "Anzahl Dermatologen Deutschland KBV" (prescriber base)
3. "Dupixent Nutzenbewertung AMNOG" (market access)
4. "Biologika Neurodermitis Wettbewerb Deutschland" (competition)
5. "Dupixent Verschreibungszahlen Deutschland 2023" (adoption)
6. "Neurodermitis Versorgungslücke Behandlung" (unmet need)

Each search builds on the previous to create comprehensive market picture.

---

**Note:** These are strategic guidelines, not rigid requirements. Adapt query formulation based on:
- Available data landscape for specific indication
- Recency of topic (newer drugs need different approaches)
- Research depth required (quick overview vs. deep analysis)
